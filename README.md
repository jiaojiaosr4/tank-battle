# 坦克大战 🎮

经典坦克大战网页游戏，纯 HTML5 Canvas 实现，打开即玩！

## 操作方式

| 按键 | 功能 |
|------|------|
| `W A S D` 或 `↑ ↓ ← →` | 移动坦克 |
| `J` 或 `空格` | 射击 |
| `R` | 重新开始 |

## 游戏规则

- 你是绿色坦克，出生在底部；红色是敌方坦克，从顶部出现
- 红色砖墙可被子弹摧毁，灰色钢铁墙不可破坏
- 消灭所有 10 个敌方坦克即可胜利
- 有 3 条命，被敌方子弹击中扣一条命
- 被击中后有短暂无敌时间（闪烁效果）

## 快速开始

直接用浏览器打开 `index.html` 即可。

```bash
# 克隆仓库
git clone https://github.com/jiaojiaosr4/tank-battle.git
cd tank-battle

# Windows 双击或用浏览器打开
start index.html
```
