# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML/CSS/JavaScript 实现的经典坦克大战网页游戏，无需任何依赖，打开即玩！

## 🕹️ 游戏特性

- **经典玩法** — 控制坦克消灭所有敌方坦克，保护基地（🦅）
- **5个关卡** — 难度递增，地图逐渐复杂
- **多种敌人** — 不同颜色、速度、血量的敌方坦克
- **道具系统** — 击杀敌人随机掉落道具：
  - ⚡ **S（速度提升）** — 移动速度加快
  - 🔥 **R（快速射击）** — 射击冷却缩短
  - 💎 **D（双发子弹）** — 同时射出两发子弹
- **可破坏地形** — 砖墙可被子弹摧毁，钢墙坚不可摧
- **粒子特效** — 爆炸、击中效果
- **音效** — 基于 Web Audio API 的射击、爆炸音效
- **移动端支持** — 触屏虚拟摇杆和开火按钮
- **暂停功能** — 按 P 键随时暂停

## 🎯 操作方式

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` 或 `W A S D` | 移动坦克 |
| `空格键` 或 `J` | 射击 |
| `P` | 暂停 / 继续 |
| `R` | 重新开始（游戏结束时） |

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可开始游戏。

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/tank-battle.git
cd tank-battle

# 用浏览器打开
open index.html
# 或 Windows:
start index.html
```

## 🛠️ 技术栈

- 纯 HTML5 / CSS3 / JavaScript（ES6+）
- HTML5 Canvas 渲染
- Web Audio API 音效
- 响应式设计（支持桌面端和移动端）

## 📝 游戏规则

1. 消灭关卡内所有敌方坦克即可进入下一关
2. 保护地图底部的基地（🦅），基地被摧毁则游戏结束
3. 玩家有 3 条生命，全部耗尽则游戏结束
4. 每消灭一个敌人 +100 分
5. 通过全部 5 关即为胜利
